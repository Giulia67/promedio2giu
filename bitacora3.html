Clase 3

Para esta semana realizaremos la bitácora personal. Vimos interfaces y códigos para diseño web (HTML, CSS,JS)

Partes de un elemento: etiqueta-atributo=valor>contenido< etiqueta/p>

!DOCTYPE html : los tipos de documento actuaban como vínculos a un conjunto de reglas que el código HTML de la página debía seguir para ser considerado bueno. Sin embargo, hoy día es simplemente un artefacto antiguo que no importa , pero debe ser incluido para que funcione correctamente.

html-html : este elemento encierra todo el contenido de la página entera a veces se conoce como elemento de raíz (root element).

body- body : encierra todo el contenido que deses mostrar a los usuarios web que visiten la pág.

Imagenes : incrusta una imagen en la página , en la posición en que aparece. Se logra a través del atributo src (source).

Elementos : permiten especificar que ciertas partes del contenido son encabezados. Un documento HTML posee seis niveles de encabezdos, h1- h6, probablemente se llegue a usar 3 o 4.

Listas : las listas desordenadas son aquellas en las que el orden de los items no es relevante, como en una lista de compras. Son encerradas en un elemento ul (unordered list). Las ordenadas son aquellas en las que el orden sí es relevante, como en una receta. Elemento ol (ordered list).

Vinculos : los enlaces son muy importantes son los que hacen de la web para implementar un vínculo, necesitas usar la abreviatura (a). Para convertir algún texto dentro de un párrafo en un vínculo.
