Clase 5

En esta semana realizamos un quizz y lo aprendido lo pusimos a prueba con ejercicios básicos para crear una pág web, aprendimos más conceptos.

Revisión de la primera semana de la bitácora, se realizó mas teoria de:

Párrafo : si se quiere especificar que se trata de un párrafo , podrías encerrar el texto con la etiqueta de párrafo. Ejemplo : < p > Mi gato es muy gruñon < /p >.

Un atributo debe tener siempre: Un espacio entre este y el nombre del elemento (o del atributo previo, si el elemento ya posse uno o más atributos). El nombre del atributo, seguido por un signo de igual (=). Comillas de apertura y de cierre, encerrando el valor del atributo.

Añadir elementos dentro de un elemento : tambien se puede colocar elementos dentro de otros elementos esto se llama anidamiento. Ejemplo : si quieres resaltar una palabra del texto << muy >> , se puede encerrarla en un elemento strong, que significa que dicha palabra se debe enfatizar.

Aprendimos a abrir archivos y editar en Visual Code.
