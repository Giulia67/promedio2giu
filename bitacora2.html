Introducción a HTML Y CSS

QUE ES HTML?

Es un lenguaje de marcado o de marcas, compuesto por etiquetas que se emplean para crear la estructura completa de una pág web , desde la cabecera (head) hasta el pie de página (footer).

QUE ES CSS?

Es una tecnología que se emplea para darle estilo a una página web construida en HTML. Se puede hacer una separación entre el contenido y la estructura de la web y su parte visual o estética.
